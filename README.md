# CASSETTE

Retro-modern local music player web app built with Vue.js 3, Vite, Tailwind CSS v4, and Tone.js.

## Features
- Load local audio files (mp3, wav, ogg, flac)
- Customize track metadata (title, artist, description, thumbnail)
- Real-time audio effects (speed, reverb, bass boost, pitch shift)
- Live EQ frequency visualizer
- Video player for background mood
- All data persisted locally via localStorage

## Tech Stack
- **Framework:** Vue.js 3 (Composition API)
- **Styling:** Tailwind CSS v4
- **Audio:** Web Audio API + Tone.js
- **Routing:** Vue Router 4
- **Storage:** LocalStorage

## Getting Started
To start the development server:
```bash
./start.sh
```
or
```bash
bun install && bun dev
```
